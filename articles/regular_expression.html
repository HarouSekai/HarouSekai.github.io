sub
文字列を置き換える。

str = "りんごを食べる"
str.sub(/りんご/,"みかん")
=> "みかんを食べる"


match
引数に指定した文字列が、メソッドを使用した文字列に含まれているかチェックする。

str = "Hello, world"
str.match(/Hello/)
=>#<MatchData "Hallo">
md = str.match(/Hello/)
md[0]
=>"Hello"
str.match(/Good/)
=> nil

\（エスケープ文字）
「正規表現において特別な意味を持つ文字」の直前に入れることで、「正規表現において特別な意味を持つ文字」をただの文字列として扱う。

\s
半角スペース。sの後に全角スペースを入れて「\s　」とすると半角・全角スペース。

gsub
文字列内に指定した文字が複数含まれている場合、そのすべてを置換する。

tel = '090-1234-5678'
tel.gsub(/-/,'')
=> "09012345678"


パスワードに英数字8文字以上の制約

pass = 'Hoge1234"
pass.match(/[a-z\d]{8,}/i)
=> #<MatchData "Hoge1234">


[]
[]で囲まれた文字のうちいずれか1つにマッチする。

a-z
アルファベットのaからzまで

\d
数字

[a-z\d]
英数字のいずれか1つ

{n, m}
直前の文字がn回以上m回以下出現するもの

i
大文字・小文字を区別しない


メールアドレスからドメイン部分のみ抽出する

mail = 'hoge@mail.com'
mail.match(/@.+/)
=> #<MatchData "hoge@mail.com">


.
ハイフンやピリオド含むすべての英数字において、どの1文字にもマッチする。

+
直前の文字が1回以上の繰り返しにマッチする。
'aaabb'.match(/a+/)
=> #<MatchDate "aaa">


郵便番号が正しく入力されたか判定
post = "250-0044"
post.match(/\A\d{3}[-]\d{4}\z/)
=> #<MatchDate "250-0044">

{n}
直前の文字がn回出現する場合にマッチする。

\A
直後の文字を文字列の先頭に持つ場合にマッチする。

\z
直前の文字を文字列の末尾に持つ場合にマッチする。



全角かな/カナ漢字の区別
[ぁ-んァ-ヶ一-龥々ー]
full_name = "田中メアリー"
full_name.match(/\A[ぁ-んァ-ヶ一-龥々ー]+\z/)
=> <MatchDate>


passwordの英数字混合の判断
password = "T2o0k2y0o"
password.match(/\A(?=.*?[a-z])(?=.*?[\d])[a-z\d]+\z/i)
=> <MatchDate>

?=
直後に設定した文字でチェックをかけて、設定した文字が続く文字列が存在する場合、その文字列にマッチする。
name = "マイケルジョーダン"
name.match(/マイケル(?=ジョーダン)/)
=> <マイケル>

*?
直前の1文字が0回以上続くとマッチする。

Railsのモデル
with_options presence: true, format: { with: /\A[ぁ-んァ-ヶ一-龥々ー]+\z/, message: '全角文字を使用してください' } do
validates :first_name
validates :last_name
end

Railsのモデル
PASSWORD_REGEX = /\A(?=.*?[a-z])(?=.*?[\d])[a-z\d]+\z/i.freeze
validates_format_of :password, with: PASSWORD_REGEX, message: 'には英字と数字の両方を含めて設定してください'
