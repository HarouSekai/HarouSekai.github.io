  • 両辺が数値なら算術演算子
  • 左右のどちらかが数値ならば文字列連結演算子
  • ※ 文字列演算子の場合、数値の型変換は不要

String型からint型への変換 

    
      String str = "24";
      int num = Integer.valueOf(str).intValue();

他の例 

    
    String si = "42";
    String sd = "2.84";
    String sb = "false";

    int i = Integer.valueOf(si).intValue();
    double d = Double.valueOf(sd).doubleValue();
    boolean b = Boolean.valueOf(sb).booleanValue();

++/--

    前置：  ++a, --b  ... 計算前に+1/-1
    後置：  a++, b--  ... 計算後に+1/-1

キャスト演算子

    
      int number = (int)10.5;   //10
      float x = (float)1.234;   //1.234f
