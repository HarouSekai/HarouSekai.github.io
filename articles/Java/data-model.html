変数

    コンパイルエラー事例集
  • 異なる型の変数を同時に宣言
    •     int a, double b;
  • 複数の変数を同時に宣言するとき、2つ目以降の変数に型名を付ける
    •     int a, int b;
  • 同じ名前の変数を宣言
    •     int a;
    double a;
  • int型にdouble型のリテラルを代入
    •     int n = 15.432;
