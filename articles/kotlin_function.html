関数を変数に格納する

関数を変数に格納する 
  
  fun main() {
    val trickFunction = ::trick
  }
  
  fun trick() {
      println("No treats!")
  }

ラムダ式を使用した関数の定義 
  
  fun main() {
    val trickFunction = trick
    trick()
  }
  
  val trick = {
      println("No treats!")
  }
  //-> No treat!

関数をデータ型として使用する

関数の型：([input type]) -> [return type]と表される。
関数の型を明示的に指定するラムダ： 
  
  val treat: () -> Unit = {
    println("Have a treat!")
  }

戻り値に関数を返す 
  
fun [function name](): [function type] {
  //code
  return [Name of another function]
}

//**exam
fun trickOrTreat(isTrick: Boolean): () -> Unit {
  if (isTrick) {
      return trick
  } else {
      return treat
  }
}

関数を引数として別の関数に渡す 
  
val [function name] = {[parameter1], [parameter2] -> 
    [function body]
}

//*呼び出される関数
fun trickOrTreat(isTrick: Boolean, extraTreat: (Int) -> String): () -> Unit {
  if (isTrick) {
      return trick
  } else {
      println(extraTreat(5))
      return treat
  }
}

//*呼び出す関数
fun main() {
  val coins: (Int) -> String = { quantity ->
      "$quantity quarters"
  }
  val treatFunction = trickOrTreat(false)
  val trickFunction = trickOrTreat(true)
}

null値許容型の関数型 
  
(([parameters]) -> [return type])?

ラムダ式でパラメータを省略する（it の使用） 
  
  val coins: (Int) -> String = {
    "$it quarters"
  }

簡単な構文でラムダを記述する

関数にラムダ式を直接渡す 
  
  val treatFunction = trickOrTreat(false, { "$it quarters" })

後置ラムダ構文（よくある関数） 
  関数の最後のパラメータが関数である場合は、別の省略形でラムダを書くことができる。
  
funcName(param) {[Lambda expression]}

val treatFunction = trickOrTreat(false) { "$it quarters" }

repeat()関数を使用する

repeat()関数：forループを簡潔に表現する 
  
for ([iteration] in [start]..[end]) {
  //code
}

repeat(times) {[iteration] ->
    //code
}
